# Applying-unfold-to-pupil-data-to-get-deconvoluted-pupil-data-for-a-AAB-task
In this project (my bachelor thesis) I created an option to get pupil data into EEGLAB, to pass it on to unfold for deconvolution. The code is specificly written for the data of one Approach-Avoidance-Experiment (conducted by Aitana-Grasso-Cladera).
